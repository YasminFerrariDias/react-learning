# React Learning

Estudos dos fundamentos do React realizados de forma independente, cobrindo os principais conceitos em ordem progressiva — do primeiro render até hooks avançados.

## Temas Abordados

- Componentes e props
- Eventos e estado
- Condicionais e listas
- Context API
- Hooks: `useEffect`, `useRef`, `useMemo` e `useCallback`

## Como Executar

Abra qualquer arquivo `.html` diretamente no navegador — não requer instalação ou build.

> Os arquivos utilizam React, Babel e Tailwind via CDN.
