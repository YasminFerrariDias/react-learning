# React Fundamentals

Estudos dos fundamentos do React realizados a partir de cursos independentes.

Os arquivos cobrem os principais conceitos do React em ordem progressiva, do primeiro render até hooks avançados como useEffect, useRef, useMemo e useCallback.

## Como Executar

Abra qualquer arquivo `.html` diretamente no navegador — não requer instalação ou build.

> Os arquivos utilizam React, Babel e Tailwind via CDN.
